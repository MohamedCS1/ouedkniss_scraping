## Guide
- install libraries:
<pre>$ pip /or/ pip3</pre>
- Check all paths
- download chromium
- check chromium path in project (chromedriver_path = {your_path})
- install BeautifulSoup:
<pre>$ pip install beautifulsoup4 </pre>
- install webdriver manager:
<pre>$ pip install webdriver-manager </pre>
